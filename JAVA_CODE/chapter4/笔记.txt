逻辑运算符
	&&短路与
	||短路或
	！取反
	&逻辑与
	|逻辑或
	^逻辑异或 不同为真，相同为假

	短路：第一个条件为假，后面条件不执行，效率高
	逻辑：都判断

赋值运算符
	基本赋值运算符 =
	复合：+= -= *= /=
		a += b -> a = a + b

	特点：
		1.运算符从右往左
		2.赋值运算符左边只能是 变量，右边可以是变量、表达式、常量值
		3.a += b -> a = a + b
		4.复合赋值运算会进行类型转换
			byte b = 3
			b += 2；//等价 b = (byte)(b + 2);
			b ++;//等价 b = (byte)(b + 1);

		三元运算符可以转换成if-else

标识符的命名规则和规范
	标识符：Java中的各种变量、方法、类名等命名使用的字符称为标识符，自己可以起名字的地方都是标识符。
	规则：
		26字母 0-9 _或$
		不可以数字开头
		不可以使用关键字和保留字，但可以包含
		Java中严格区分大小写，长度无限制
		不能包含空格
	规范：
		1.包名：多单词组成，所有字母小写
		2.类名、接口：多单词组成时，所有单词首字母大写-大驼峰
		3.变量名、方法名：多单词组成时，第一个单词首字母小写，其余大写-小驼峰，驼峰法
		4.常量名：所有字母大写，多单词用下划线连接
		5.看文档

位运算
	>>算数向右 低位溢出，符号位不变，用符号位补溢出的高位
	<<算数向左 符号不变，低位补0
	>>>无符号右移 逻辑右移 低位溢出，高位补0
	&按位与
	|按位或
	~按位取反
	^按位异或

原码反码补码
	1.二进制最高位是符号位0正1负
	2.正数三码合一
	3.负数反码 符号位不变，其他取反
	4.负数补码 取反+1
	5.0 反码补码都是0
	6.Java中没有无符号数，Java中的数都是有符号的
	7.计算机运行都是用补码方式运算

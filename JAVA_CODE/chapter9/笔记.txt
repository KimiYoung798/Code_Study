房屋出租系统

    需求：
        能够实现对房屋信息的添加，修改和删除，用数组实现，并打印明细表

    设计：
        分层模式，  当软件比较复杂，需要模式管理

        //界面层
        HouseView -- 显示界面，接受用户输入，调用其他类完成对房屋信息的操作
        //业务层
        HouseService -- 响应house view调用，完成对房屋信息的各种操作CRUD。
        //domain/model模型层
        House -- 一个house对象就是一个房屋信息
        //程序入口
        HouseRentApp -- 创建HouseView，调用该对象，显示主菜单
        //工具类
        utility
